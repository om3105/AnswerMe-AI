# API Documentation

> Full API documentation will be generated in Step 8.
> Interactive docs are auto-generated by FastAPI at `/docs` (Swagger) and `/redoc` (ReDoc).

## Base URL

```
Development: http://localhost/api/v1
Production:  https://api.answerme.ai/v1
```

## Authentication

All protected endpoints require a Bearer token in the Authorization header:

```
Authorization: Bearer <access_token>
```
